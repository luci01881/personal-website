# Modern Personal Portfolio

A sleek, responsive, and high-performance personal portfolio website built with clean, semantic HTML5, modern CSS3 (Custom Properties & Flexbox/Grid), and vanilla JavaScript. Designed with a dark mode aesthetic tailored for developers and technical professionals.

## 🚀 Live Demo

You can view the live deployment of this portfolio here:
👉 **https://luci01881.github.io/**

## ✨ Features

- **Modern Dark Aesthetic:** Clean interface focused on readability and user experience.
- **Fully Responsive:** Perfectly optimized for desktops, tablets, and mobile screens.
- **Interactive Mobile Menu:** Seamless navigation experience on smaller viewports using vanilla JS.
- **Smooth Scrolling:** Easy navigation between sections via the fixed header.
- **Clean Codebase:** Zero external framework dependencies, no clutter, and production-ready.

## 🛠️ Built With

*   **HTML5** - Structured semantic markup.
*   **CSS3** - Custom variables, layout alignment (Flexbox/Grid), and media queries.
*   **JavaScript** - Lightweight interaction automation.
*   **FontAwesome** - Modern vector icons for social media and details.
